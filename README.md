add-vagrantuser Cookbook
=============
Adds a Vagrant User for a Vagrant Box that lacks one (e.g. Softlayer boxes)

Attributes
----------
* None Yet
    - nothing to tune

Usage
-----

Just include chef.add_recipe "add-vagrantuser" to your Vagrantfile.
